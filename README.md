# 🌐 Curso Completo de Desenvolvimento Web — Do Zero ao Profissional 🚀

<br>

> Aprenda Desenvolvimento Web completo dominando **HTML5**, **CSS3**, **Bootstrap**, **JavaScript**, **PHP**, **MySQL**, **jQuery**, **APIs REST**, **MVC**, **WordPress** e muito mais.
>
> Ideal para quem deseja se tornar um desenvolvedor Full Stack através de projetos reais e aplicações completas.

<br>

---

<br>

## 🎯 Objetivo do Curso

<br>

Este curso foi criado para transformar iniciantes em desenvolvedores web completos, cobrindo desde a criação de páginas estáticas até sistemas dinâmicos integrados com banco de dados, APIs e aplicações mobile.

<br>

---

<br>

## 🧭 Estrutura do Curso

<br>

O conteúdo está organizado em módulos progressivos para facilitar o aprendizado.

<br>

### 🐣 Nível Iniciante

- Introdução ao Desenvolvimento Web
- Estrutura de páginas com HTML5
- Estilização com CSS3
- Responsividade básica
- Conceitos fundamentais da Internet

<br>

### ⚙️ Nível Intermediário

- Bootstrap para interfaces responsivas
- JavaScript moderno (ES6+)
- Manipulação do DOM
- Eventos e validações
- AJAX e consumo de APIs

<br>

### 🧱 Nível Avançado

- PHP Orientado a Objetos
- Arquitetura MVC
- APIs REST
- Integração com bancos de dados
- Desenvolvimento Mobile com Ionic
